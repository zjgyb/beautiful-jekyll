# 数值的拓展

## 全局方法的改变
`Number.parseInt` & `Number.parseFloat`
```
// es6把以前的es5的模式进行了改进，更加注重模块化的方式
// true
Number.parseInt === parseInt
Number.parseFloat === parseFloat 
```
{: .box-note}
## 判断是否为整数
方法：`Number.isInteger`

```
Number.isInteger(37); // true
Number.isInteger(37.5); // false
```
{: .box-note}
## 新增了一些Math方法（有对数双曲函数等）
## 新增了指数运算符

```
2 ** 5 // 32
```
